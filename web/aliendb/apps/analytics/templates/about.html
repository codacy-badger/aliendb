About AlienDB

AlienDB is an analytics platform for trending Reddit threads. It stores historical data for the top threads on Reddit, including karma trends and comment stats. This website is not affiliated with Reddit, and only accesses data which is publicly available.

AlienDB monitors the top 100 submissions on /r/all and periodically records information including each submission's rank, its number of comments, and its current karma. Additionally, each comment is analyzed using the Natural Language Toolkit (NLTK) and TextBlob for statistics such as polarity, subjectivity, gold given, and special user presence. Comment text is not stored.

Statistics from individual submissions are also used to provide subreddit averages. It is important to note that only submissions which reach the top of Reddit are included in these averages, so subreddit averages should not be used as a way to describe a subreddit as a whole. Rather, it is a useful tool to determine whether an individual submission is significantly different than others which were also highly upvoted from the subreddit.
